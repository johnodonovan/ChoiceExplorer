ChoiceExplorer

This is a repository for our ChoiceExplorer project. A domain-independent interactive tool for visualizing and exploring recommendations based on any corpus of user profiles. For example, visualizing collaborative filtering of course selections in a university, or items in a shopping cart. It works with any collection of profiles!
